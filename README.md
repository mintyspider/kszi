# Мой Kotlin проект

## Как использовать

```powershell
# Собрать проект
.\build.ps1

# Запустить
.\run.ps1

# Или одной командой из корневой (этой) папки
.\br.ps1
```

# Структура проекта
src/main/kotlin/ - исходный код

build.ps1 - скрипт сборки

run.ps1 - скрипт запуска

README.md - отчёт

# Создание и исполнение проекта
